Binary Search Trees
// Created from AI Chat Session · Tagged: DSA
Definition
A Binary Search Tree (BST) is a node-based binary tree data structure that follows the BST property: every node's left subtree contains only nodes with val < node.val, and the right subtree contains only nodes with val > node.val.
Key Property: Left child < Parent < Right child — at every level of the tree.
Time Complexities
  • Search: O(log n) average · O(n) worst case
  • Insertion: O(log n) average · O(n) worst case
  • Deletion: O(log n) average · O(n) worst case
Insertion Algorithm (Python)
class Node:
  def __init__(self, val):
    self.val = val
    self.left = self.right = None

def insert(root, val):
  if not root: return Node(val)
  if val < root.val: root.left = insert(root.left, val)
  else: root.right = insert(root.right, val)
  return root
Traversal Orders
  • In-order (L→N→R): Produces sorted output ✅
  • Pre-order (N→L→R): Used for tree copying
  • Post-order (L→R→N): Used for deletion
